# nodets-pet
Projeto para estudo/portifólio UseSteph - Node + Typescript

### Pré-requisitos globais:
`npm install -g nodemon typescript ts-node`

### Utilizando variáveis de ambiente para configurar a PORT e a Conexão com Banco

### Instalação
`npm install`

### Executar o projeto 
`npm run start-dev`
